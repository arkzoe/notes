## 算数运算
- 支持字面量和变量
- 字面量
	- 正数
	- 小数
- 加减乘除
- 简化
- 除法-分数到小数转换丢失
- ++z减少临时变量
## 生命周期
- 定义时占用，出{}释放
- {}嵌套子作用域
```c++
int main(){
	int x;
	{
		//子作用域
	}
}
```
## 常量
- const--运行时常量
	- 可以用变量初始化
- constexpr--编译时常量
## auto
- 确保变量被初始化
- 代码简洁、泛型用
- 推导常量会变为普通变量
- 用于强制转换
