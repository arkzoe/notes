	- 复制对象内存
- 深拷贝
	- 复制对象内存
	- 复制对象成员指向的堆空间
- 转换构造函数
	- 参数当前类的对象引用
# 移动语义
1. 右值引用
	- 左值
		- 变量
		- 可取地址
		- 即使变量的类型是右值引用，表达式仍是左值表达式
	- 右值
		- 不能取地址
	- 右值引用
		- Type&&var
2. move
	- 普通对象强制转换成右值引用
	- 移动后源对象失效
3. 移动构造
	- String(String&& s)
4. 智能指针
	- 可移动构造/赋值
	- 不满足可复制构造/赋值
	- 支持管理单个对象和对象数组
	- unique_ptr
	- make_unique
		- 产生更小更快的代码
		- 减少重复类型设置
		- 内存安全
	- 修改--reset
	- 释放控制权--release
	- 移动
