# 程序生成过程各文件作用

| 源码       |        |
| -------- | ------ |
| .h       | 引用，不编译 |
| .cpp     | 编译     |
| .sln     | 解决方案   |
| .vcxproj | 项目配置   |
| 目标文件     |        |
| .obj     | cpp编译后 |
| 执行文件     |        |
| .exe     | 多obj合并 |
# 程序生成过程
预处理->编译->汇编->链接->执行
- 预处理
	- .cpp
- 编译--把cpp代码转化成目标机器系统的汇编代码
	- .s
- 汇编--编译为机器指令
	- .obj
- 链接--将多个obj合并，并设置程序入口
	- .exe
- 执行
