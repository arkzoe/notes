| 函数        | 作用                          |
| --------- | --------------------------- |
| find()    | 检测子串是否包含在这个字符串中，在返回下标，否返回-1 |
| index()   | 报异常                         |
| replace() | 返回替换后的字符串                   |
| split()   | 返回切割后的列表序列                  |

- 列表[]--动态数组
	- 名称=[1,2,3,4,....]
	- for遍历
	- 增
		- append增加指定数据
		- extend追加
		- insert指定位置
	- 改：下标
	- 查：in，not in
		- index
		- count
		- 统计小串在大串中出现次数
			- count
			- split切割+统计
			- 替换：(大串长度-新串长度)/小串长度
			- find+切片
	- 删：del，pop，remove移除第一个匹配项，clear
	- 排序：sort，reverse
- 元组()--不可变数组
	- index，count，len
- 字典{键值对}--哈希表
	- 查
		- get(key，默认值)：根据key返回value，不存在返回第二参数，若省略返回None
		- keys()：以列表返回一个字典所有的键
		- values()：以列表返回字典的所有值
		- items()：以列表返回可遍历（键，值）的元组数据
	- 删：del
- 集合{单值，}--hashset
- 冻结集合frozenset--不可变哈希集合

	- 集合
