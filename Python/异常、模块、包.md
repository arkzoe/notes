# 异常
```python
try:
	可能发生错误的代码
except Exception as e:
	print(e)
	出现异常执行
else:
	没引发错误执行
finally:
	必执行
```
- 异常具有传递性
	- 主函数设置异常捕获
# 模块
- 一个py就是一个模块
- 引入
	- import 模块名
	- from 模块名 import 功能名
	- from 模块名 import*
	- import 模块名 as 别名
	- from 模块名 import 功能名 as 别名
- 自定义：符合标识符命名规则
- 多模块有同名功能，调用后导入模块的功能
```python
__name__属性
当前模块是__main__
调用者是模块名
if __name__ == '__main__'

__all__ = ['']
使用from 模块名 import*导入，只能导入这个列表的数据
```
