# 值类型和引用类型
### 值类型
- 存在栈中
	- 内存小，速度快
- 结构体
- 整型
- 浮点型
- bool型
### 引用类型
- 存在堆中
	- 内存大，速度慢
- 字符串：string
- 数组：Array
- 自定义：类，接口，委托
- object
out
- 使用变量的地址
- 使用数组或结构体返回多个数据
- 被out修饰的参数在函数内部需要被赋值
- 调用时，out参数要加out
ref
- 使用变量地址
- 直接修改实参的值
- 变量用ref进行传参，必须先赋值
- 调用时，ref参数前加ref
# 结构体
- 不能有无参构造
- 对所有参数赋值
### 可变参数-params
- 参数个数可变
- 将实参列表中跟可变参数数组类型一致的元素当作数组元素处理
- 只能作为参数列表中最后一个参数
```c#
static void t(params int[] arr){

}
```

- 函数重载
