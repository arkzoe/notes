| 8        | sbyte   | byte   |

| 浮点类型 |         |        |

| 字符     | 布尔    | 字符串 |

## 变量名规范
- 数字、字母、下划线组合
- 字母、下划线开头
- 不能用关键字
- 不能重名
- 驼峰
## 常量
- const--静态常量
- readonly--动态常量
	- 只能声明为类字段
	- 声明/构造函数初始化，初始化后无法修改
# 运算符
- 算数
- 自增自减
- 赋值
	- ==
	- /=
	- +=
	- ...
- 关系
- 三元----？xx : xx
- 逻辑
	- !
	- && 短路与 左false 右不执行

## 字符串格式化
- +
- 占位符，{0}，第0个占位符
## 转义字符
```c#
\' 单引号
\" 双引号
\\ 反斜杠
\0 空字符
\a 警告音
\b 退格
\n 换行
\r 置换
\t 水平Tab
\v 置换Tab
```
# 显式转换
- Convert-一个类型转成另一个类型
- Parse-转成字符串--失败抛异常
- TryParse-不抛异常-返回bool
- 自定义
