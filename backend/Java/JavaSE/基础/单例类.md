# 单例设计模式
- 作用：确保某个类只能创建一个对象
## 写法
- 把类的构造器私有
- 定义一个类变量记住类的一个对象
- 定义一个类方法，返回对象
### 饿汉式单例类
- 拿对象时，对象早已创建
```java
public class A {
	//2、定义一静态变量，基于本类的一个唯一对象
	private static A a = new A();//用 finnal , 不用提供方法

	//1、私有构造器
	private A(){
	}

	//3、定义一个类方法返回对象
	public static A getObject(){
		return a;
	}
}
```
### 懒汉式单例类
- 用对象时，才开始创建对象
```java
public class B {
	//2、定义一个类变量存储对象
	private static B b;
	
	//1、私有化构造器
	private B(){
	}
	
	//3、提供静态方法返回对象，真正需要时创建
	public static B get(){
		if(b == null){
			b = new B();
		}
		return b;
	}
}
```
