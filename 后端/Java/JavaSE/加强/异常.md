# 异常
![[异常体系.png]]
## 基本处理
#### 抛出异常（throws）
- 方法内部的异常抛出给调用者
```java
方法 throws 异常1、异常2、异常3...{
	...
}
```
#### 捕获异常(try...catch)
- 直接捕获
```java
try{
	//监视可能出现异常的代码
}catch(异常类型1 变量){
	//处理异常
}catch(异常类型2 变量){
	//处理异常
}...
```
## 作用
- 定位bug关键信息
- 作为方法内部的特殊返回值，通知调用者，方法执行问题   //项目能继续运行
```java
throw new RuntimeException()
```
