# 异常
![[异常体系.png]]
## 基本处理
#### 抛出异常（throws）
- 方法内部的异常抛出给调用者
```java
方法 throws 异常1、异常2、异常3...{
	...
}
```
#### 捕获异常(try...catch)
- 直接捕获
```java
try{
	//监视可能出现异常的代码
}catch(异常类型1 变量){
	//处理异常
}catch(异常类型2 变量){
	//处理异常
}...
```
## 作用
- 定位bug关键信息
- 作为方法内部的特殊返回值，通知调用者，方法执行问题   //项目能继续运行
```java
throw new RuntimeException()
```
## 自定义异常
- 两类

| 自定义运行时异常                                   | 自定义编译时异常                            |
| -------------------------------------------------- | ------------------------------------------- |
| 定义一个异常类继承RuntimeException.                | 定义一个异常类继承Exception                 |
| 重写构造器                                         | 重写构造器                                  |
| 通过thrownew异常类（xxx）来创建异常对象并抛出      | 通过thrownew异常类（xxx）创建异常对象并抛出 |
| 编译阶段不报错，运行时才可能出现！提醒不属于激进型 | 编译阶段就报错，提醒比较激进                |
