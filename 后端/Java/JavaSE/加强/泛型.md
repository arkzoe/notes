	-  泛型类、泛型接口、泛型方法
```java
public class ArrayList<E>{
	...
}
```
- 编译时约束数据类型，自动进行检查
	- 避免强制类型转换

## 泛型方法、通配符、上下限
### 泛型方法
```java
修饰符 <类型变量、类型变量，...> 返回值类型 方法名（形参列表）{

}
public <T> void test(T t){

}
```
### 通配符
- “？”，使用泛型时代表一切类，E、T、K、V等定义泛型时使用
### 泛型上下限
- 上限：?extends Car:   ?  接收必须是Car或者其子类
- 下限：?super Car:       ?  能接受的必须是Car或者其父类
### 泛型支持类型
- 不支持基本数据类型，只能支持对象类型（引用数据类型）
	- 泛型擦除：泛型编译后没用了，编译后擦除，所有类型恢复成Object
	- 万物皆对象 -> 包装类：基本数据类型包装成对象类型
		- valueOf 提前封装好对象，节省内存
		- 自动装箱：基本数据类型自动转为包装类型
		- 自动拆箱：包装类型自动转为基本数据类型
