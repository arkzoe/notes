继承父类非私有成员
```
            子类       父类 
public class B  extends A  {

}
```
## 权限修饰符

| private   | 本类                       |
| --------- | -------------------------- |
| 缺省      | 本类、同一个包中的类       |
| protected | 本类、同一个包中的类、子类 |
| public    | 任意                       |
==private== < 缺省 < protected < ==public==

## 特点
- 单继承
- 多层继承
- 祖宗类
- 就近原则--先去最近的父类找
- ==super==--指定访问父类 
## 方法重写
- 名称一样，参数一样
- ==@Override== 重写的校验注解
- 私有、静态不能重写
- 声明不变，重新实现
### 应用场景
- 重写 toString() ，返回对象内容
	- 直接输出对象会调用 Object.toString() 输出地址
## 子类构造器
- 先调用父类构造器在调用自己的
- 必须先调用父类的
	- 默认构造器第一行都是 super() ，会调用父类无参构造器
	- 先初始化父类
### this() 调用兄弟构造器
构造器重载
默认参数
super()，this() 必须写第一行不能同时出现
