
| 修饰  |                         | 场景  |
| --- | ----------------------- | --- |
| 类   | 最终类，不能被继承               | 工具类 |
| 方法  | 最终方法，不能被重写              |     |
| 变量  | 该变量有且仅能被赋值一次            |     |
|     | 基本类型：储存数据不能被改变          |     |
|     | 引用类型：储存地址不能被改变，地址指向内容可变 |     |
- 变量
	- 成员变量
		- 静态
			- 常量：被final修饰，存储固定值
				- 常量名称大写，多个单词下划线连接
				- 记录系统配置信息
				- 常量包
				- 编译后常量被“宏替换”，性能和直接使用字面量一样 
		- 实例
			- final修饰一般没有意义
	- 局部变量
