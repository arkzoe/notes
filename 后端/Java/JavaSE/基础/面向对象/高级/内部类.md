- 一个类定义在另一个类的内部
## 成员内部类
- 类中的一个普通成员，类似普通成员变量、成员方法
- 无static修饰，属于外部类的对象持有
外部类名称.内部类名称 对象名 = new ==外部类名称().new 内部类名称()==
### 访问其他成员的特点
1. 成员内部类中可以直接访问外部类的静态成员、实例成员
2. 直接拿到当前寄生的外部类对象  外部名.this
## 静态内部类
 - static修饰的内部类，属于外部类自己持有
外部类名称.内部类名称 对象名 = new ==外部类名称.内部类名称()==
### 特点
1. 成员内部类中可以直接访问外部类的静态成员，==实例成员不可以==
2. 直接拿到当前寄生的外部类对象  外部名.this

## ==匿名内部类==
- 匿名：不需要声明，默认有隐藏的名字
- 实际上有名字，外部类名+$.class
- 本质是一个子类，并会立即创建出一个子类对象
- 更方便的创建一个子类对象
```java
> new 类或接口--已存在(参数值...){

	类体（方法重写）
}
```
### 常见形式
- 作为一个对象参数传输给方法
- 对象回调
### 应用场景
- 调用别人提供的方法实现需求，传输匿名内部类对象给其使用
- 排序sort，比较器
