# 目录结构
- 无盘符，只有一个根目录/
# 基础
- **格式**
```shell
# 格式
# 命令名 选项 参数
command [-options] [parameter]
```
- **ls**
```shell
# list 查看路径下所有子集
# all 所有，line 行，human 人性化
ls       # 查看当前目录子集，不包括隐藏的，等价于：ls ./
ls -a    # 查看当前目录子集，包括隐藏的
ls -al   # 以运行方式，查看当前目录子集，包括隐藏的，需结合h使用
ls -alh  # 运行，以人性化方式，查看当前目录子集，包括隐藏的
ls /     # 查看根目录所有字子集，不包括隐藏的
ls -l /  # 以运行方式，查看根目录子集，不包括隐藏的
ll /     # 同上，ll等价于ls -l
```
目录切换
- **cd**
```shell
# change directory,改变目录
cd       # 不写参数，回家，root账号目录 /root，其他账号 /home
cd /
```
- **pwd**
```shell
# print work directory,查看当前所在的工作目录
pwd
```
- 相对路径：以当前目录为起点
- 绝对路径：以根目录为启点
- **特殊路径符**
	- .    ：表示当前目录
	- ..   ：表示上一级目录
	- ~   ：表示home目录
创建目录
- **mkdir**
```shell
# make directory,创建目录，在home目录，否则权限不足
mkdir [-p] 路径
-p 可选，自动创建不存在的父目录
```
文件操作
- **touch**
```shell
# 创建文件
touch 路径
```
- **cat**
```shell
# 查看文件，将内容全部显示出来
cat 路径
```
- **more**
```shell
# 查看文件，支持翻页，内容过多可以一页页展示
more 路径
```
- **cp**
```shell
# copy，复制文件/文件夹
cp [-r] 参数1 参数2
# -r 复制文件夹使用
# 参数1：被复制的文件
# 参数2：目的地
```
- **mv**
```shell
# move 移动文件，文件夹/改名
mv 参数1 参数2
# 参数1：被复制的文件
# 参数2：目的地

```
- **rm**
```shell
# remove 移动文件文件夹
rm [-r -f] 参数1 参数2...
# -r 删除文件夹
# -f 强制删除
#    普通用户删除内容不会弹出提示，只有root有提示
# 参数1 参数2...要删除的文件或文件夹
```
- rm支持通配符，及匹配任意内容
查找命令
- **which**
```shell
# 查找命令的程序文件，Linux命令
which 要查找的命令
```
- **find**
```shell
# 查找指定文件
find 起始路径 -name "被查找的文件"
# 结合通配符做模糊查询
find 起始路径 -size +|- n[kMG]
# +、-大于和小于
# n 大小数字
# kMG表示大小单位，k表示kb，M表示mb，g表示gb

```
过滤
- **grep**
```shell
# 从文件中通过关键字过滤文件行
grep [-n] 关键字 文件路径
# -n 在结果中显示匹配的行的行号
# 关键字，表示过滤的关键字，带有空格或其他特殊符号，用""包
# 文件路径，可视作内容输入端口
```
统计
- **wc**
```shell
wc [-c -m -l -w] 文件路径
-c 统计bytes数
-m 统计字符数
-l 统计行数
-w 统计单词数
```
- 管道符：|
	- 将管道符左边命令的结果作为右边命令的输入
- **echo**
```shell
# 在命令行输出指定内容
echo 输出的内容，""包裹
``` 
- 反引号/飘号
```shell
`` 包裹的内容被当作命令执行
```
- 重定向符
```shell
>  将左侧命令的结果，覆盖写入到右侧指定的文件中
>> 将左侧命令的结果，追加写入到右侧指定的文件中
```
- tail
```shell
# 查看文件尾部内容，跟踪文件最新更改
tail [-f -num] 路径
# -f 持续跟踪
# -num 查看尾部多少行，默认10
```
- vi/vim编辑器
	- 命令模式
		- 所敲的按键理解为命令，以命令执行不同的功能’
		- 不能进行文本编辑
	- 输入模式
		- 编辑模式，插入模式
		- 文本编辑
	- 底线命令模式
		- 以：开始，用于文件保存、退出
![[vim.png]]
```shell
vim 文件路径
# 文件不存在，编辑新文件
# 存在，编辑已有文件
```

| 模式  | 命令  | 描述             |
| --- | --- | -------------- |
| 命令  | i   | 在当前光标位置进入输入    |
| 命令  | a   | 在当前光标位置之后，进入输入 |
| 命令  | I   | 在当前行的开头        |
| 命令  | A   | 当前行的结尾         |
| 命令  | o   | 当前光标下一行        |
| 命令  | O   | 当前光标上一行        |
| 输入  | esc | 回到命令           |
![[命令模式快捷键.png]]

| 底线命令模式    |        |
| --------- | ------ |
| wq        | 保存并退出  |
| q         | 仅退出    |
| q!        | 强制退出   |
| w         | 仅保存    |
| set nu    | 显示行号   |
| set paste | 设置粘贴模式 |
